## 点击引用之前请必看

Media部分规则 由 Me （不要脸的） 从网上搜刮整理而来，在提供特定 流媒体服务使用指定策略或节点以供在QX中ZhuangBi的同时，也提供了适用于 **懒人** 的单一文件规则。
